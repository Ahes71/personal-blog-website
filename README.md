# 🌐 Personal Blog Website

A modern and responsive blog website built using HTML, CSS, and JavaScript.
This project focuses on clean design, smooth user experience, and interactive UI elements.

## 🚀 Features

* 📱 Fully responsive design (mobile-friendly)
* ✨ Animated custom cursor effect
* 🔄 Smooth scrolling navigation
* 🎨 Clean and modern UI/UX

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📌 Project Purpose

This project was built to improve my frontend development skills and practice creating modern, user-friendly web interfaces.

## 🚧 Future Improvements

* Dark mode support
* Blog post system (dynamic content)
* Backend integration
